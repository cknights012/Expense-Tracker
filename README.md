# Expense-Tracker

A simple Java console application to track daily expenses. Users can add expenses, view all expenses, and calculate the total amount spent.

## Features

* Add new expenses
* View all expenses
* Calculate total expenses
* Menu-driven console interface

## Technologies Used

* Java
* JDK 17 (or above)
* ArrayList
* Scanner Class

## Project Structure

* Expense.java - Stores expense details.
* ExpenseTracker.java - Main application with all functionalities.

## Sample Features

1. Add Expense
2. View Expenses
3. View Total Expense
4. Exit
